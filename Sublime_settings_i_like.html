Sublime settings i work on

After installing sublime for the 1st time, i usually make a couple of configurations that i find interesting.

Install the following addons:

In order to install addons to sublime you start by installing package manager addon. To do so, you press Ctrl+Shift+P and type "install", select the option "Package Controll:install package control" and the package management suite will be ready to use.

From now on , to install an addon you can use the same menu, and select "install package", a list of available packages will show. From there i use the ones listed below
Emmet: Emmet for Sublime Text File Icons: Adds icons on the sidebar, differenting each file type Side​Bar​Enhancements: adds right click menu options on sidebar SublimeCodeIntel: Full-featured code intelligence and smart autocomplete for Sublime Text. Sublime Linter:The code linting framework for Sublime Text 3. No linters included JSLint:Javascript Linter Bootstrap Autocomplete:Bootstrap snippets and autocmplete AdvancedNewFile: adds a new file option on the right click of sidebar Material-Theme

The next think to do is to change default font and theme settings to make reading easier. I have found the following settings to work for me :

"font_face": "Fira Code",
"font_size": 10,
"line_padding_bottom": 2,
"line_padding_top": 2,
"margin": 0,
"material_theme_compact_panel": true,
"material_theme_compact_sidebar": true,
"material_theme_contrast_mode": true,
"material_theme_small_statusbar": true,
"material_theme_small_tab": true,
"material_theme_tabs_autowidth": true,
"theme": "Material-Theme-Darker.sublime-theme"
